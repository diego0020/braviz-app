#Braviz
Interactive Brain Data Visualization
## Install:
Add the location of this directory to PYTHONPATH environment variable.
Prerequisites:

- Python >2.75
- VTK >6.0 with Python wrapping

Python packages:

- nibabel
- colorbrewer
- numpy
- scipy
- rdflib*
- httplib2*
- psutil*
- matplotlib
- seaborn
- rpy2 (for anova, linear model)
- tornado (for parallel coordinates)
